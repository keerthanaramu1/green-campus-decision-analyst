- MongoDB (Upcoming)

## ⚙️ Installation Steps

### 1️⃣ Clone Repository

git clone https://github.com/keerthanaramu1/green-campus-decision-analyst.git

### 2️⃣ Navigate to Server

cd green-campus-project/server

### 3️⃣ Install Dependencies

npm install

### 4️⃣ Create .env File

PORT=5000

### 5️⃣ Run Server

node index.js

Server will run on:
http://localhost:5000
